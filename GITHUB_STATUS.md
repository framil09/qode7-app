## 🚀 Próximos Passos

### 1. Acessar o Repositório
Acesse: https://github.com/framil09/qode7-app

### 2. Deploy no AWS Amplify
Siga o guia: [AMPLIFY_QUICK_START.md](./AMPLIFY_QUICK_START.md)

**Passos rápidos:**
1. Acesse https://console.aws.amazon.com/amplify/
2. Clique em "New app" → "Host web app"
3. Selecione GitHub
4. Escolha o repositório: `framil09/qode7-app`
5. Selecione o branch: `main`
6. O Amplify detectará automaticamente o `amplify.yml`
7. Configure variáveis de ambiente
8. Clique em "Save and deploy"

### 3. Configurar Variáveis de Ambiente
No AWS Amplify Console, adicione:
```env
NODE_ENV=production
NODE_VERSION=20
```
